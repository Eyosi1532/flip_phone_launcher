# Flip Phone Launcher

A **custom Android launcher app** built with **Flutter/Dart** that converts smartphones into a minimalist “flip phone” style interface.  

## Purpose

The Flip Phone Launcher was created to help users **reduce distractions from social media and overuse of smartphones**. By restricting access to only essential apps and providing a simple, retro-style interface, this app encourages **digital minimalism** and **better focus**.

---

## Features

- **Rotating circular app dial** — apps are arranged in a circular layout that can be rotated to select the desired app.
- **Essential apps only** — Phone, Messages, Camera, Clock/Alarm, Settings, Calendar, Calculator.
- **Customizable background image** — dark overlay ensures text and icons are readable.
- **Minimalist UI** — reduces eye strain and promotes focus.
- **Interactive gesture handling** — smooth rotation and app selection.

---

## Technologies Used

- **Flutter & Dart** for cross-platform mobile development.
- **device_apps package** to interact with installed Android applications.
- Responsive layout for different screen sizes.
- Custom UI elements including circular app containers and rotation gestures.
